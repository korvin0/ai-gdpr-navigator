Вот полная архитектура бизнес-логики для пилотной версии (MVP) AI GDPR Navigator.
  Интеграция Google Sheets: Чтение 4-х вкладок (Logic, Content, Triggers, Hints).
  FSM (Машина состояний):
    • State 0: Логический квест по схемам (GDPR Status). Если GDPR не применяется — завершение без аудита.
    • State 1: Опрос по триггерам (профилирование + роли: is_creator, is_brand_owner, is_modifier).
    • State 2: Интерактивный список мер с кнопками «Сделано/Инфо».
    • State 3: Итоговый отчет + чат с Gemini API.
  Логика подсказок: Приоритет отдается тексту из Google Sheets. Gemini подключается только в самом конце или по спец. кнопке.
  Анонимность: Хранить прогресс только в текущей сессии (chat_id), без регистрации.
