Фаза 3: Завершение, Экспорт и Консультация (Gemini AI)
Это финальный этап, где бот превращает пройденный чек-лист в осязаемый результат для бизнеса и открывает окно для общения с ИИ по тем вопросам, на которые не хватило стандартных подсказок из таблицы.

1. Генерация финального статуса
Когда пользователь просмотрел все блоки в Фазе 2, бот подводит итог.
    • Логика: Бот сравнивает общее количество обязательных пунктов (согласно фильтрам) с количеством нажатых кнопок «✅ Сделано».
    • Сообщение:
«Аудит завершен! 🏁 Выполнили: 18 пунктов. Пропустили: 5 пунктов.
Ваш текущий уровень комплаенса: 78%»

2. Экспорт отчета (MVP-формат)
Поскольку в пилоте мы не храним данные в БД, бот должен мгновенно выдать результат, который пользователь сможет сохранить.
Варианты выгрузки:
    1. Текстовый summary: Бот присылает длинное сообщение со списком «Что нужно доработать» (все пункты со статусом Todo или Skipped).
    2. Генерация PDF (опционально): Разработчик использует библиотеку (напр., FPDF), чтобы собрать аккуратный файл с логотипом, датой и списком мер.

3. Подключение Gemini AI: Режим «Экспертный совет»
Вот теперь, когда основной путь пройден, мы открываем доступ к Gemini. Это идеальный момент: пользователь уже погружен в контекст и у него появились специфические вопросы.
Бизнес-логика функции «Спросить эксперта»:
    • Точка входа: Кнопка «💬 Задать вопрос эксперту» в финальном меню.
    • System Prompt (Контекст): Бот отправляет Gemini скрытый текст:
"Ты — эксперт по защите данных и AI Act. Твоя задача — помочь разработчику внедрить меры из чек-листа CNIL. Проект пользователя имеет следующие параметры: [is_child=True, is_gen_ai=False...]. Ответ должен быть технически точным и соответствовать GDPR."
    • Диалог: Пользователь пишет, например: "У меня нет возможности переобучить модель, как мне иначе реализовать право на удаление данных?".
    • Ответ Gemini: ИИ анализирует ввод и дает рекомендации (например, про фильтрацию вывода или дообучение на пустых весах), опираясь на лучшие практики.

4. Финальное меню (Loopback)
Чтобы пользователь не чувствовал себя брошенным, бот предлагает:
    • 🔄 Начать новый аудит (сброс всех флагов сессии).
    • 📢 Поделиться ботом (ссылка на бота).
    • 📝 Оставить отзыв.
